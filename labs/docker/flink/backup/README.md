```
./build.sh --from-release --flink-version <x.x.x> --scala-version <x.xx> --hadoop-version <x.x> [--image-name <image>]

E.g.,

./build.sh --from-release --flink-version 1.8.0 --scala-version 2.11 --image-name flink-s3
```
